The DNN application is handwritten number recognition,
the images are 28x28=784 pixel images.

-The dataset folder contains files to generate the data for training, validation
and testing of the DNN. They are extracted from the MNIST package.

-The sigmoid folder contains files to generate the sigmoid LUT, to load into the DNN.

-The testbenching folder contains files to compare the testbench output with the
output of python scripts

-The weights and bias folder contains files to generate the DNN parameters.
