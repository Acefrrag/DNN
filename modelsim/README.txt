The "tb_files" folder contains the parameters to be loaded into the DNN, together with the inputs to be fed into it.

The "test" folder contains the testbench files.

The "scripts" folder contains python script to train the DNN and generate fixed-point representation data to be loaded into the architecture.

The "sim" folder contains modelsim simulation projects to visualize the data waveforms.

The "src" folder contains the VHDL design files.
